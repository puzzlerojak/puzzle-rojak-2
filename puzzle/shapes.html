How do I start this puzzle? This is a mangled clues puzzle https://devjoe.appspot.com/huntindex/keyword/mangledclues
Each line is a crossword-style clue, but all the words have been transformed in some specific way. You need to figure out what each transformation was by trying to understand what the original clued sentence could have been.
We solved the clues, now what? After solving each crossword clue, use the same transformation you observed on the clue to transform your answers. This will give you a new set of answers which can be paired with the shapes. If you're having trouble relating them, try writing out the shape names.
